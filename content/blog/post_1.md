We had a lot of fun finally announcing Directory in public this week and getting our first handful of users! You all had lots of great feedback for us to work on! The beta version of Directory is available to use in your browser right now for North Carolina residents! 

I wanted to start this blog as an informal way to be more transparent about whats happening with Directory. So here's a little info on where we're heading.

### Roadmap

**Some of our top priorities right now**:

- Open source all of our code for transparency and to open up a public contribution process.
- Perfect and finish the base features. For example, we still need to add:
    - location services
	- Notifications
    - "hidden" tag section
	- saved search filter
	- Blocking
- Expand our Durham, Chapel Hill, and Raleigh user base
- Release native apps for iOS and Android
    - We are mostly waiting on our DUNS number to get assigned to the LLC
		  so we can create developer accounts with Google and Apple. And this takes about 30 days.
- Have stable funding for the base expenses of keeping the service running.

**Some priorities down the road**:
- Become a nonprofit with tax-exempt status
- Expand to other cities
- Have funding for moderators and maintainers.
