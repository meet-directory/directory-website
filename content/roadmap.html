Roadmap


Directory is available to use in your browser right now!

We're hoping to release native iOS and Android apps in about a month.

But in the meantime, Directory works best on desktop in any browser that isn't Safari. It currently also works on mobile Android browsers.


Our plan in order of priority

  1. Open source all of our code
  2. Perfect and finish base features
  3. Release native version for both iOS and Android
  4. Get nonprofit tax-exempt status
  5. Expand our Durham user base
  6. Expand to other cities
